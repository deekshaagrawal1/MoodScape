# MoodScape 🌈

## 📌 Project Overview
Moodscape is a web application that provides personalized content based on a user’s current mood. Users can select or search their mood (such as happy, sad, or bored), and the app dynamically displays relevant quotes and activity suggestions using public APIs.

---

## 🎯 Purpose
The goal of this project is to create an engaging and user-friendly platform that connects emotional states with meaningful content. It enhances user experience by delivering relevant suggestions instead of generic recommendations.

---

## 🔌 APIs Used
The application will integrate the following public APIs:

- **Quotable API (Primary API):** Fetches inspirational and mood-based quotes  
- **Bored API:** Provides activity suggestions based on user interest  

---

## ✨ Planned Features
- Mood Selection → Users can choose or search their current mood  
- Dynamic Content → Display quotes and activities based on mood  
- Search Functionality → Search content using keywords  
- Filtering → Filter results by type (quotes or activities)  
- Sorting → Sort results based on relevance or length  
- Favorites → Save preferred items using local storage  
- Responsive Design → Works across mobile, tablet, and desktop  
- Dark/Light Mode (optional)

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Fetch API for API integration  
- Array Higher-Order Functions (map, filter, sort, etc.)

## ⚙️ Setup Instructions

Follow these steps to run the project locally:

1. Clone the repository:
   git clone https://github.com/deekshaagrawal1/MoodScape.git
2. Navigate to the project directory:
   cd MoodScape
3. Open the project:
   - Locate the `index.html` file  
   - Double-click it or open it using any web browser  
4. The application will run in your browser.
